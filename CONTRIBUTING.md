# Contributing 
Pour contribuer à ce projet, vous devez suivre ce lien:[ #eiffel_game2:libreti.net](https://matrix.to/#/#eiffel_game2:libreti.net)

Vous devrez vous créer un compte Matrix si vous n'en avez pas déjà un. Aussi, il vous faudra installer Element ou utiliser la version web pour que Matrix fonctionne correctement. Une fois que vous êtes connecté et que vous avez rejoins le groupe de discussion, vous n'avez qu'à exprimer votre envie de participer au projet à Louis Marchand, le développeur d'EiffelGame2.
